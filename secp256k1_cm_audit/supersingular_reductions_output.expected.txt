---- Supersingularity criterion for j = 0 curves ----

Theorem (Deuring): for ℓ ≠ 2, 3, the curve y² = x³ + b over
F_ℓ is supersingular iff ℓ ≡ 2 (mod 3), regardless of b.

Equivalently: -3 is a non-residue mod ℓ  ⇔  ℓ is inert in Z[ω]
            ⇔  the j=0 curve over F_ℓ is supersingular.

---- Connection (or lack thereof) to F_p ECDLP ----

For each supersingular reduction E_ℓ of y² = x³ + 7:

• End(E_ℓ) is a maximal order in the quaternion algebra
  B_{ℓ,∞} = the unique quaternion algebra over Q ramified
  exactly at ℓ and ∞.

• j(E_ℓ) = 0 always (since the equation y² = x³ + 7 has j = 0).

• The supersingular j-invariant set over F̄_ℓ contains roughly
  ℓ/12 elements; j = 0 is one of them when ℓ ≡ 2 mod 3.

• Castryck-Decru-Maino-Martindale-Panny-Pope-Robert (2022)
  broke SIDH by exploiting the supersingular ℓ-isogeny
  problem with auxiliary torsion data.  This is a SUPERSINGULAR-
  WORLD attack, not connected to F_p ECDLP.

---- Why supersingular reductions DON'T attack F_p ECDLP ----

Deuring's lifting theorem: an ordinary curve over F_p lifts
uniquely (up to F_p²-isomorphism) from any of its reductions.
BUT the lifting is FROM F_ℓ TO characteristic 0, then specialise
BACK to F_p.  This DOES NOT give a useful 'attack pipeline'
because:

  (a) The lift takes you to characteristic 0 (Q-bar / Q_p), not
      to a 'smaller' DLP problem.

  (b) The supersingular isogeny graph at ℓ has rich structure,
      but its DLP-relevant features (e.g., GMP-isogeny-problem
      hardness, SIDH-CSIDH) are about the ISOGENY problem on
      the graph, not about ECDLP on individual curves in the
      graph.

  (c) Castryck-Decru's SIDH break needs specific torsion data
      that's published by the protocol.  For our setting (we
      just have an ECDLP instance on E/F_p), no such data is
      published.

  (d) The j-invariant 0 supersingular curve over F̄_ℓ is the
      same algebraic-geometric object as secp256k1's reduction,
      but reading ECDLP information from this object back to
      F_p requires lifting through Deuring, which loses the
      DLP secret in the process.

---- Smart's attack revisited via canonical lift ----

Smart 1999 attack on anomalous curves (n = p) uses the
CANONICAL LIFT of E/F_p to a curve E/Z_p over the p-adic
integers, then applies the formal logarithm to extract the
discrete log.

For secp256k1: n ≠ p (n = p + 1 - t with t ≈ 2^{129} ≠ 1).
Therefore canonical-lift extraction of the discrete log via
the formal logarithm DOES NOT WORK — the formal log produces
a multiple of n in Z_p, but extracting d from this requires
knowing n / gcd(n, p) which equals n.  Then d · log(P) =
log(Q) mod p², which gives d mod (something) but not d mod n
when n ∤ p.

So canonical lifts give nothing for non-anomalous curves like
secp256k1.  Confirmed null.

V4 verdict: supersingular-reduction direction is null for
           secp256k1's F_p ECDLP.
================================================================

• Supersingular reductions of secp256k1 exist for every prime
  ℓ ≡ 2 (mod 3) (≈ half of all primes); each is in the
  j=0 supersingular point of F̄_ℓ.

• These reductions live in the SIDH/CSIDH supersingular world,
  but the supersingular ℓ-isogeny problem (and its Castryck-
  Decru-style breaks) are mathematically disjoint from F_p
  ECDLP.  No known reduction in either direction.

• Canonical-lift attacks (Smart 1999) require anomalous curves
  (#E = p) — secp256k1 is not anomalous.

⇒ V4 direction is closed for prime-field ECDLP on secp256k1.
  The supersingular isogeny graph is a rich mathematical
  object with its own cryptanalytic story (SIDH break) but
  does not attack F_p ECDLP on ordinary curves.
